Stability Levels

Please note the following stability levels. Each CSS property in this project is assigned a level describing its stability; some values or pseudo-classes under a property might possibly have their own level, in which case this would override the property’s stability level.

For example, the group of classes .-d (corresponding to the CSS property display) might have Stability 4, but .-d-fx (for display: flex) could have Stability 3, and .-d-il-nK (for display: inline on extra-small screens) could have Stability 2.

  1. Stability 0:ObsoleteThis feature is no longer supported. Do not use it.

  2. Stability 1:DeprecatedBreaking changes are planned for this feature, or it will be removed in future versions. It is no longer in development, and should not be relied upon.

  3. Stability 2:ExperimentalThis feature is in development, but is unsupported or subject to change. It could be removed in future versions.

  4. Stability 3:StableThis feature has been tested and proven satisfactory. It is in development, but no breaking changes will be made unless absolutely necessary.

  5. Stability 4:LockedOnly patches will be made to this feature. It is no longer in development. It may be heavily relied upon and will not be removed in future versions.
